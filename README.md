# Low Frequency Audio Amplifier

This project presents the design and simulation of a low-frequency audio amplifier using bipolar junction transistors (BJTs).

The amplifier was designed as part of an Analog Electronics course project and simulated using OrCAD PSpice.

## Features
- Multi-stage BJT amplifier design
- DC operating point analysis
- Circuit simulation in OrCAD PSpice

## Tools
- OrCAD PSpice
- Analog circuit design
